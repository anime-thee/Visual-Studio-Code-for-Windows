# What is Visual Studio Code?

Visual Studio Code (VS Code) is a free, cross-platform source code editor designed for modern software development. Unlike full-featured integrated development environments (IDEs), VS Code remains lightweight while still offering intelligent coding features, integrated debugging, version control, terminal access, and thousands of extensions.

The editor is suitable for web development, desktop software, AI projects, machine learning, cloud computing, DevOps, game development, and embedded programming.
